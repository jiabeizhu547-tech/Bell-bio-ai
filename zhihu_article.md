## 我做了什么

### 1. 蛋白质二级结构预测（Q3 89.8%）

输入一串氨基酸（比如 `MKVLILACLVALALA...`），输出每个残基是 α-螺旋（H）、β-折叠（E）还是无规卷曲（C）。

就像给蛋白质做了个"CT 扫描"，能看到它的局部形状。

| 模型 | Q3 准确率 | H F1 | E F1 | C F1 |
|------|----------|------|------|------|
| 从零训练 CNN+BiLSTM | 85.5% | 0.887 | 0.780 | 0.854 |
| Meta ESM-2 微调 | 89.2% | 0.925 | 0.846 | 0.879 |
| **集成 + 生物规则平滑** | **89.8%** | **0.930** | **0.850** | **0.885** |

几个有意思的发现：

- **ESM-2（800 万参数）完胜从零训练的 CNN-BiLSTM**。预训练的知识迁移效果远超模型容量的影响。
- **β-折叠最难预测**（F1 0.85），因为它涉及远距离肽链段的排列，单看局部序列很难推断。
- **集成只提升了 0.5%**，因为 ESM-2 已经足够强，V1 能提供的互补信息有限。

### 2. EC 酶功能分类（准确率 87.3%）

不仅看结构，还能判断这条蛋白质属于哪类酶：

| EC 类别 | 例子 | F1 |
|---------|------|-----|
| 氧化还原酶 | 乙醇脱氢酶 | 0.86 |
| 转移酶 | 激酶 | 0.83 |
| 水解酶 | 胰蛋白酶 | 0.85 |
| 裂合酶 | 碳酸酐酶 | 0.81 |
| 异构酶 | 磷酸葡萄糖异构酶 | 0.90 |
| 连接酶 | DNA 连接酶 | 0.88 |
| **转位酶** | **ATP 合酶** | **0.97** 🏆 |

数据来自 UniProt 数据库的 **33,331 条**真实酶序列。方法很简单——用 ESM-2 提取序列嵌入，然后用一个仅 **266 KB** 的小 MLP 分类。

## 技术架构

```
蛋白质序列 "MKVLILAC..."
    │
    ├─→ ESM-2 (6层 Transformer, 800万参数)
    │       ├─→ 分类头 → Q3 二级结构 (H/E/C)
    │       └─→ Mean Pooling → MLP → EC 酶类别 (1~7)
    │
    └─→ CNN+BiLSTM (从零训练, 180万参数)
            └─→ 集成投票 → 生物规则平滑 → 最终预测
```

全部在 **CPU** 上训练（感谢 Meta 把 ESM-2 做得这么轻量）。

## 可复现的关键经验

如果你也想做蛋白质 AI，这几点可能有用：

1. **站在巨人肩上**。ESM-2 最小的 8M 模型就够用，不需要 GPU 集群。
2. **数据质量 > 模型复杂度**。从 UniProt REST API 爬真实酶数据花了最多时间，但 87.3% 的准确率证明值得。
3. **生物规则后处理有效**。最小片段约束（H/E 至少 3 个残基连续）让预测更符合物理规律。
4. **中文 AI 生态**。ModelScope 提供免费 GPU + Gradio 部署，对国内开发者很友好。
